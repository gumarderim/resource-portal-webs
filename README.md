# Resource Portal Webs

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源的集中存放与版本管理使用。

## 项目简介

本仓库旨在提供一个结构化的 HTML 页面存储空间，方便对多个独立页面进行统一管理、版本追踪与静态发布。每个页面均可独立访问，互不依赖，适合用于存放各类信息展示页、工具页或临时页面。

## 目录说明

```
resource-portal-webs/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 可选的总览页面
└── README.md       # 本文件
```

- `pages/`：每个页面一个子目录或独立文件，命名清晰，便于识别。
- `assets/`：存放多个页面共用的样式、脚本或图片资源，避免重复。
- `index.html`：可选的入口页面，用于快速导航至各子页面。

## 页面归档说明

- 每个页面均为完整的独立 HTML 文件，包含必要的样式与脚本（或引用公共资源）。
- 页面内容遵循标准 HTML 规范，兼容主流浏览器。
- 新增页面时请放入 `pages/` 目录，并保持命名规范。
- 如需更新已有页面，请通过 Pull Request 提交，并注明变更内容。

## 维护说明

- 欢迎提交 Issue 反馈问题或建议。
